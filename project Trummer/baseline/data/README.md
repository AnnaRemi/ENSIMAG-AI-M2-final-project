# Data provenance

`all_reviews.csv` was copied verbatim from:

https://github.com/itrummer/llmjoins/blob/main/testdata/all_reviews.csv

The official repository uses this labeled 100-review sample for the IMDb
experiment. The paper cites the larger source dataset:

https://www.kaggle.com/datasets/atulanandjha/imdb-50k-movie-reviews-test-your-bert

Columns:

```text
text,sentiment
```

`sentiment` is `neg` or `pos`.

The following files are generated by `prepare_data.py`:

- `reviews_1.csv`
- `reviews_2.csv`
- `same_reviews_ground_truth.csv`
