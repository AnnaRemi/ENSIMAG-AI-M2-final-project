# Full fixed implementations

The complete source code for the four solutions is available in the following directories and matching archives under `code_bundles/`:

1. `project SUQL/baseline/` — SUQL baseline.
2. `project SUQL/v1/` — SUQL two-level cascade.
3. `project Trummer/baseline/` — Trummer adaptive block join baseline.
4. `project Trummer/v1/` — Trummer structured two-level cascade.

The shared benchmark and Aker execution code is in `benchmarks/`. It is also included in `code_bundles/benchmark_and_aker_tools.tar.gz`.

Generated Python caches, macOS metadata, model outputs, and benchmark result files are not included in the archives.
